# Violet Void Theme for aerc

A dark purple-tinted theme for [aerc](https://aerc-mail.org/) email client.

## Install

```bash
mkdir -p ~/.config/aerc/stylesets
cp violet-void-theme ~/.config/aerc/stylesets/

# In your aerc.conf:
# styleset-name=violet-void-theme
```
